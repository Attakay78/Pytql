    Base Model

    The base model for creating all Table Models. Inherite the base model to create a new Table Model.

    class pytql.model.Model

            Bases: object

    Example

    class Student(Model):
    first_name = CharField(name="First Name", max_length=20)
    last_name = CharField()
    age = IntField()
    length = IntField()
